### 利用脚手架创建命令

npm install -g create-react-app // 全局安装 create-react-app (只需要安装一次)

create-react-app 【自己起一个项目名称】 // 创建项目

cd 【自己起的项目名称】 // 进入项目目录

### `npm run dev `
